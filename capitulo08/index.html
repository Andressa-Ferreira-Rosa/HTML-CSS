Exercício 5 (cap. 8)

Semântica

Devemos sempre estar de olho nas atualizações do html, pois vez ou outra uma tag se torna obsoleta. Para isso podemos consultar o site W3.org, o site official da Worldwide Web.

Vale mencionar que neste capítulo nos foi apresentada a tag address, que como sugere, ela entrega, além de um significado, a formatação adequada a um endereço.

Negrito e itálico

Negrito / destaque

Nesta frase temos um termo em destaque porem usando a tag b(não semântica). Nesta frase também temos um termo em destaque, porém usando a tag strong(semântica).

Itálico / ênfase

Nesta frase temos um termo em itálico, usando a tag i(Não semântica). Nesta frase também temos um termo em itálico, porém usando a tag em(semântica).

Outras formatações

Texto marcado

Neste texto há um termo marcado usando a tag mark.

Atenção: a tag mark ainda não foi reconhecida pelo VSCode.

Texto grande e pequeno

Nesta frase temos um termo grande(usando a tag big, que é obsoleta e não semântica) e um pequeno(usando a tag small, que é semântica por ser usada em contratos).

Texto deletado

Podemos destacar um texto como excluído para indicar que ele deve ser lido mas não considerado, usando a tag del.

Texto inserido

Podemos marcar um texto como inserido para indicar que ele foi adicionado posteriormente, usando a tag ins.

Atenção: embora ins e u proporcionem o mesmo efeito visual, a tag ins é semântica, diferentemente da tag u, que não é semântica.

Sobrescrito e subscrito

Nesta frase temos um texto sobrescrito e um texto subscrito, usando as tags sup e sub, respectivamente. Exemplos de uso: 22=4 e água= H2O.

Código-fonte e pré-formatação

O código document.getElemntById('teste') é usado em JavaScript. Esse código foi formatado na tag code.

Já o código a seguir foi formatado tanto na tag code quanto em pre. Essa última tag garante o espaçamento e a tabulação exatamente igual ao digitado no VSCode.

    
        num= int(input('Digite um número:'))
        if num % 2 == 0
            print(f'O número {num} é par')
        else:
            print(f'O número {num} é impar')
        print('Fim do programa')
