Links para download

Este é um exemplo de link para download de um pdf. Um link para download é compostopela tag a+download+type, sendo que em download pomos o nome do arquivo e em type pomos o tipo de mídia que queremos baixar.

Se você quer saber o que escrever dentro do atributo type de uma âncora de hypertext, consulte a lista oficial da IANA.org.
